# Skills Taxonomy (Kernel AI Assistant)

This repository organizes skills by domain:

- `skills/ebpf/` - eBPF diagnostics and observability workflows
- `skills/kubernetes/` - pod/network/policy and cluster debugging
- `skills/kernel/` - kernel build, patch, regression, bisect workflows
- `skills/distro/` - distro-specific package/setup compatibility checks
- `skills/embedded/` - Yocto/Buildroot/cross-compile flows
- `skills/xdp/` - XDP packet filtering and high-performance networking
